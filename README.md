# История FFmpeg и патентных войн

Коллекция motion graphics сцен для документального фильма о Fabrice Bellard, истории FFmpeg и патентных войнах вокруг медиакодеков.

**→ [Открыть](https://agasyants.github.io/FFmpegVisualization/)**

---

## Что это

Каждая страница — отдельный анимационный сегмент, снятый с экрана как видеофутаж. Рендер в реальном времени прямо в браузере, без экспорта и внешних зависимостей.

Темы: GIF/Unisys, MP3, MPEG, DVD-лицензии, DivX, OGG Vorbis, раскол FFmpeg/Libav, Fabrice Bellard.

## Структура

| Папка | Название | Содержание |
|---|---|---|
| `PatentWar/` | АД | Патентные войны: GIF, MP3, MPEG, DVD, DivX, OGG |
| `SpacePart/` | КОСМОС | История FFmpeg и Беллара |
| `ToUse/` | ВЫ | Практическое использование FFmpeg |

## Управление

Общее для большинства сцен:

| Клавиша | Действие |
|---|---|
| `R` / `К` | Перезапуск |
| `Пробел` | Следующая сцена |
| `F` | Zoom-режим (таймлайн) |
| `Q` | Автовоспроизведение |
| `Tab` | Скрыть UI |

## Технический стек

- HTML5 Canvas 2D и WebGL
- Рендер 2560×1440, 60 fps
- Чистый JS, без фреймворков
- Шрифты: Google Fonts (Oswald, IBM Plex Mono, Bebas Neue и др.)
